NEaB Chat Next (NChatN)

This is the home for NChatN. If you haven't played Nowhere-Else and Beyond then you probably won't really care much about this little project. If you do play NEaB then you may want to take a quick look at this chat client, because it offers some cool improvements over the standard chat client.

Overview

NChatN should have fewer bugs (at least fewer visible bugs), be more efficient, and should be able to emulate most features of the regular NEaB chat. There are currently a few missing features that I may add in the future, but if they're something you rely on, you may be better off with the old chat. They are:

  • No tab completion box is shown (tab completion for usernames works, however the box that says "Tab for: " doesn't show up)
  • There is no "Away" button (you can use the /away or /afk commands though`)
  • Tab completion of usernames and adding smileys through the smiley box have a different way of spacing compared to the old chat (this is my preference, however I may add a configuration option for it later)

Features

Some of the features that set NChatN apart from the standard NEaB client are:

  • View some of what's been said in chat before you entered
  • Better username Tab-completion (can complete usernames in the middle of a sentence)
  • Keeps a full record of what's been said in chat since you opened the window
  • Allows you to easily save all of what's been said to your computer
  • Better scrolling support (no longer jerks down when someone says something in chat, and keeps your position when resizing the window)
  • Visual indicator shows when scrolling up into chat history
  • Ability to have more than four chat channels open at once (not that anyone needs that ability with NEaB's chat)

Installation

Installing NChatN has become much easier since the switch to a GreaseMonkey/TamperMonkey script. However, it still has a few steps you have to go through. Please note that this chat is only tested in the latest versions of Firefox and Chrome! If you would like to help make it work in another browser, jump down to the "Contributing" section.

Firefox

  1. Install GreaseMonkey
  2. After you have GreaseMonkey installed simply click on the link right here and install the script
  3. Load (or reload) NEaB's chat window

Chrome

  1. Install TamperMonkey
  2. After you have TamperMonkey installed click on the link right here and select the option to install it as a TamperMonkey script (important step!)
  3. Load (or reload) NEaB's chat window

Updating

There should be no need to update anything. The script is setup to update all on its own when a new release is available!

Rules

I have spent quite a bit of time trying to make sure that NChatN follows the rules and guidelines of NEaB, as well as preventing an easy vector for abuse or other rule-breaking action. If you find a bug that allows for such a thing to take place, please file a bug report here about it immediately or let me know in chat!

Similarly, I will not accept any code contributions that exploit game bugs or are contrary to the game's spirit.

Release Notes

Version 2.0

This version brings about a few very fundamental changes to the way NChatN functions. This also forces its deviation from the original function of the original NEaB chat client, so you may have to learn to do things a bit differently with NChatN now. Changes:

  • Whispers are now shown in tabs instead of inline!
    • The tab will display "W: " to indicate that it is a whisper-tab instead of a regular channel
  • The online player list has changed appearance and function considerably
    • Player names turn gray when they go away (in addition to the old "Away" sign over their avatar)
    • A fancy new (animated!) menu appears when you click on a player in a non-whisper channel
    • The player name dropdown isn't there anymore, instead click on the player you want to whisper to and select "Private Chat"
    • In order to get the player info for someone in chat click on the player you want to view the info of and select "Whois"
  • Channel tabs look a bit different now
    • Edges are rounded and a drop shadow has been applied to the text
    • The selected tab appears darker and is a larger appearing tab now
    • Channels with very long names are now displayed properly (The name is clipped, and hovering over the tab shows the full name, i.e. "Very Long Channel Name" would be something like "Very Long Cha..." and hovering over the tab would show the full name)
    • Because so many tabs can be open now (becuase of whisper in tab) the tab area should automatically show a scrollbar when there are too many tabs to fit on the screen
    • Closing tabs that you're not currently in no longer places you in a non-existant channel
  • The chat interface has gotten slimmer and cleaner
    • The header is smaller (the connection status light has moved into the "Messages" bar)
    • All elements should be pixel-perfect aligned with other elements
    • The scroll-up indicator (the blue glow at the bottom of chat when you scroll up into chat history) should now fit in the chat window properly
  • Added a new popup that appears when NChatN has updated
    • This feature can be turned off by going to Menu -> Settings -> Update Messages

Issues:

  • Whisper tabs only show players name in the online list no titles or avatars and clicking on a player's name brings up the player info window, not the new fancy menu
  • Still no custom user scripts allowed (Sorry about that)
