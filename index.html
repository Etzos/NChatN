NEaB Chat Next (NChatN)

This is the home for NChatN. If you haven't played Nowhere-Else and Beyond then you probably won't really care much about this little project. If you do play NEaB then you may want to take a quick look at this chat client, because it offers some cool improvements over the standard chat client.

Overview

NChatN should have fewer bugs (at least fewer visible bugs), be more efficient, and should be able to emulate most features of the regular NEaB chat. There are currently a few missing features that I may add in the future, but if they're something you rely on, you may be better off with the old chat. They are:

  • No tab completion box is shown (tab completion for usernames works, however the box that says "Tab for: " doesn't show up)
  • Usernames aren't made "clickable" in the chat window (however, you're free to use the /who <username> and /whois <username> commands)
  • There is no "Away" button (you can use the /away or /afk commands though`)
  • Tab completion of usernames and adding smileys through the smiley box have a different way of spacing compared to the old chat (this is my preference, however I may add a configuration option for it later)

Installation

Installation has become much easier since the switch to a GreaseMonkey/TamperMonkey script. However, it still has a few steps you have to go through. Please note that this chat is only tested in the latest versions of Firefox and Chrome! If you would like to help make it work in another browser, jump down to the "Contributing" section.

Firefox

  1. Install GreaseMonkey
  2. After you have GreaseMonkey installed simply click on the link right here and install the script
  3. Load (or reload) NEaB's chat window

Chrome

  1. Install TamperMonkey
  2. After you have TamperMonkey installed click on the link right here and select the option to install it as a TamperMonkey script (important step!)
  3. Load (or reload) NEaB's chat window

Updating

There should be no need to update anything. The script is setup to update all on its own when a new release is available!

Rules

I have spent quite a bit of time trying to make sure that NChatN follows the rules and guidelines of NEaB, as well as preventing an easy vector for abuse or other rule-breaking action. If you find a bug that allows for such a thing to take place, please file a bug report here about it immediately or let me know in chat!

Similarly, I will not accept any code contributions that exploit game bugs or are contrary to the game's spirit.
