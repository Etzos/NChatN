You can use the bookmarklet method which only works on Firefox.
Or, you can use the shim script for GreaseMonkey (Firefox) or Tampermonkey (Chrome).

If you're using Tampermonkey the process is more complicated (Google is a pain in some cases).

  1. Click on the Tapermonkey icon and select "Add a new script..."
  2. Clear out all of the text in the entry box there
  3. Come back to this page
  4. Click and drag the shim script link back into the entry box you just cleared
  5. Press the save button.
