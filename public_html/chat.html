Start Chat [Commands]
To chat with other players, simply enter your message and press enter!

Chat supports the following commands:
/look <username> Look at a player in chat.
/kick <username> Kick another player.
/who <username> See the player"s stat card.
/me <username> <action> Do something as yourself in chat.
/msg <username> <message> Whisper to someone in chat privately.
/r <username> <reply> Reply to a player's whisper in private.
/online <username> List online players.
/hug <username> Hug another player.
/store <username> <message> Store a message for a player. The player will see it when he/she opens chat. This is displayed publically.
/private <username> <message> Does the same thing as /store, but will be displayed in private.
/help Request help. Have you tried checking the Wiki first?
/yawn Yawn.
/afk Go afk (away from keyboard).
/away Does the same as /afk.
/report Send a report to the admins about a player that is misbehaving in chat.

If the username contains a space replace it with an underscore (_).
For example, the name 'Black Orchid' would become 'Black_Orchid'.
